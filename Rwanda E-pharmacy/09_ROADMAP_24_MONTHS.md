# 24-Month Roadmap

## Months 1-2: Validation

- Interview patients, pharmacies, doctors, insurers, and regulators.
- Validate core pain points.
- Confirm compliance assumptions.
- Build investor-grade clickable demo.
- Select pilot district.

## Months 3-4: MVP Foundation

- Auth and roles.
- Medicine master database.
- Pharmacy registration and verification.
- Inventory management.
- Patient search.
- Prescription upload.
- Reservation workflow.

## Months 5-6: Pilot Launch

- Onboard 20-50 pharmacies.
- Launch public search.
- Launch pharmacy dashboard.
- Monitor stock quality.
- Collect patient feedback.
- Produce first impact report.

## Months 7-9: Insurance and Delivery

- Add insurance coverage rules.
- Add pharmacy-insurance mapping.
- Add delivery request flow.
- Add delivery SOP.
- Add chronic refill reminders.

## Months 10-12: Analytics and Revenue

- Add government analytics dashboard.
- Add pharmacy Pro reporting.
- Add ethical ads module.
- Add institutional dashboard pilot.
- Prepare grant and investor reporting.

## Months 13-18: Regional Scale

- Expand to more districts.
- Add multilingual support.
- Add API integrations for larger pharmacies.
- Improve stock forecasting.
- Improve prescription extraction.
- Add distributor and wholesaler workflows.

## Months 19-24: National Readiness

- Harden infrastructure.
- Formalize institutional partnerships.
- Add advanced analytics.
- Run security and compliance audits.
- Create national operating model.
- Prepare larger funding round or national partnership proposal.

## Build Priorities

Do first:

- Verified pharmacy stock.
- Medicine search.
- Reservation.
- Prescription upload.
- Admin verification.

Do next:

- Insurance rules.
- Delivery.
- Analytics.
- Ads.

Do later:

- Deep hospital integrations.
- Claims processing.
- AI forecasting.
- National interoperability.
